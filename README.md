# 📚 Student Management System

A full-stack **MERN (MongoDB, Express, React, Node.js)** application for managing student records with a clean UI built using **Tailwind CSS**.

---

## 🚀 Live Demo

- 🌐 Frontend: https://
- ⚙️ Backend API: https://

---

## 🧑‍💻 Features

- ➕ Add new student
- 📄 View all students
- ❌ Delete student
- 📱 Fully responsive UI (Mobile + Desktop)
- ⚡ Instant UI update (no refresh needed)
- 🎨 Clean UI using Tailwind CSS

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository
```bash
git clone https://github.com/Mayank4161/StudentManagementSystem.git

2️⃣ Backend Setup
cd Backend
npm install
Create .env file:
PORT=5000
MONGO_URI=your_mongodb_connection_string

Run backend:
npm start

3️⃣ Frontend Setup

cd Frontend
npm install
npm run dev
